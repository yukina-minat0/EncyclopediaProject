AI 万物百科 (Ultimate Edition)
这是一个基于 PyQt6 开发的多模态智能交互系统，集成了先进的计算机视觉算法与大语言模型，旨在提供像素级精准识别与“萌娘百科”级别的深度知识获取体验。

核心逻辑流
系统采用 “预判-识别-重试-深度合成” 的闭环架构：

目标提取：利用 FastSAM 实现点选或框选区域的实时图像分割。

智能分流：由 Qwen3-VL 预判目标属性。

二次元目标：调用专用识别引擎，并内置 3 次错误重试与 Vision 兜底机制。

现实物体：直接由 Vision 引擎进行高精度工业/常识识别。

深度百科：接入 DeepSeek-V3，通过“萌娘百科”提示词工程，合成包含萌点、设定、梗与技术参数的深度干货。

多模态播报：支持多种音色实时朗读，并允许用户针对识别结果进行上下文追问。

技术栈
前端交互：PyQt6 (Python)

视觉算法：FastSAM (Segment Anything), Qwen3-VL-Flash

语言模型：DeepSeek-V3 (deepseek-chat)

音频处理：Pygame (TTS 播报控制)

图像处理：OpenCV, Torch
